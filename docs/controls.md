# Controls

## Mouse

| Action | Effect |
|--------|--------|
| Click | Place pixel |
| Scroll | Zoom in/out |
| Drag (left button) | Pan the map |
| Shift + Drag | Draw mission rectangle (alliance leaders) |

## Keyboard

| Key | Effect |
|-----|--------|
| 1-9 | Switch to palette color 1-9 |
| Tab (hold) | Eyedropper — pick color from map |
| Arrow keys | Pan the map |
| W / A / S / D | Pan the map |

## Palette Rebinding

You can rebind number keys to different colors:

1. Hold **Ctrl** and press a **digit** (1-9)
2. Click on any swatch in the palette
3. That digit is now bound to that color

Bindings persist across sessions (saved in localStorage).

## Replay Mode

- Click **Replay** in the nav bar to enter replay mode
- Use the scrubber to rewind/fast-forward through the map's history
- Click **Exit** or press **Escape** to return to live mode
- During replay, the map shows only pixels that existed at that point in time
